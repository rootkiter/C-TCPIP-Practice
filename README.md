### Windows  
윈속(윈도우 소켓)을 기반으로 프로그램을 개발하기 위해서는 기본적으로 다음 두 가지를 진행해야 한다.
- 헤더파일 winsock2.h를 포함한다.  
- ws2_32.lib 라이브러리를 링크시켜야 한다.  
  - 프로젝트 단위 링크 방법: 프로젝트 '속성' - '구성 속성' - '링커' - '입력' - '추가 종속성' - 'ws2_32.lib' 추가.
